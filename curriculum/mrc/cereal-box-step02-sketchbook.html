Elements of Art Sketches
Seven elements. Seven sketches. Show each one clearly.
SKETCHING CHALLENGE / 01 / 05
Make the Seven Elements Stick
You read the Elements of Art page and watched the seven short element videos. Now your hand has to learn them. The fastest way is to sketch each one.
You will make seven sketches, one for each element: Line, Shape, Form, Color, Value, Texture, Space. You can put a few on the same sheet of paper if you like.
This is not a perfect masterpiece. This is a fast study. Focus on showing what each element means.
HOW TO COMPLETE THIS ASSIGNMENT / 02 / 05
Three Steps to Finish
Read the Elements of Art page first. Then sketch all seven on your sketch paper. Then take clear phone photos and upload one PDF.
01
Review the Elements
Open the Elements of Art page and watch the seven element videos before you start. Keep the page open as you sketch.
02
Sketch All 7 Elements
Do all seven elements. You can group a few on the same sheet if you like. Spend a few focused minutes on each. Add short notes that label your drawing.
03
Photograph + Submit
Take a clear phone photo of each sketch and combine them into one PDF. Upload it to this Canvas assignment.
YOUR SEVEN SKETCHES / 03 / 05
What to Draw for Each Element
Here is a quick idea for what to draw for each element. You can choose your own subject. The goal is to show the element clearly.
Element 01
Line
Draw lots of different lines: straight, curvy, thick, thin, dotted, jagged. Label a few.
Element 02
Shape
Fill an area with flat shapes: circles, squares, stars, blobs. Keep them flat. No shading.
Element 03
Form
Take a shape and make it 3D with shading. A flat circle becomes a sphere. A square becomes a cube.
Element 04
Color
Make a color wheel or paint big patches of warm and cool colors. Label which group is which.
Element 05
Value
Draw a row of boxes that go from white to black. Show the steps in between. This is a value scale.
Element 06
Texture
Sketch four boxes. Fill each with a different texture: smooth, rough, fuzzy, bumpy. Label each one.
Element 07
Space
Draw one strong object in a mostly empty space. Show how the empty space makes the object pop.
WHAT I LOOK FOR / 04 / 05
Expectations
Clear Element
Each sketch shows the element it is supposed to show. No guessing.
Neat and Intentional
Sketches look like you cared. No scribbles. No fillers.
Short Notes
A line or two on each sketch explains how your drawing shows the element.
All 7 In Class
All seven sketches get done in class. Color is welcome if it helps your idea.
WHAT TO SUBMIT / 05 / 05
One Clean PDF
Take a clear phone photo of each sketch and combine them into one PDF. Make sure every sketch is bright, in focus, and easy to read. Upload the PDF here.
Work fast. Stay focused. This is about understanding, not making a masterpiece. When you finish, click the Done button to move to Step 03.
