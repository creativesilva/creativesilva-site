SKETCHBOOK CHALLENGE / 01 / 05
Make the Seven Elements Stick
You read the Elements of Art page. Now your hand has to learn them. The fastest way is to sketch each one.
You will fill seven sketchbook pages, one element per page: Line, Shape, Form, Color, Value, Texture, Space.
This is not a perfect masterpiece. This is a fast study. Focus on showing what each element means.
HOW TO COMPLETE THIS ASSIGNMENT / 02 / 05
Three Steps to Finish
Read the Elements of Art page first. Then sketch all seven in your sketchbook. Then scan and upload one PDF.
01
Review the Elements
Open the Elements of Art page from this module before you start. Keep it open as you sketch.
02
Sketch All 7 Pages
One element per page. Spend a few focused minutes on each. Add short notes that label your drawing.
03
Scan to One PDF + Submit
Scan all 7 pages into one clear PDF. Upload it to this Canvas assignment.
YOUR SEVEN SKETCHES / 03 / 05
One Element Per Page
Here is a quick idea for what to draw on each page. You can choose your own subject. The goal is to show the element clearly.
Page 01
Line
Draw lots of different lines: straight, curvy, thick, thin, dotted, jagged. Label a few.
Page 02
Shape
Fill the page with flat shapes: circles, squares, stars, blobs. Keep them flat. No shading.
Page 03
Form
Take a shape and make it 3D with shading. A flat circle becomes a sphere. A square becomes a cube.
Page 04
Color
Make a color wheel or paint big patches of warm and cool colors. Label which group is which.
Page 05
Value
Draw a row of boxes that go from white to black. Show the steps in between. This is a value scale.
Page 06
Texture
Sketch four boxes. Fill each with a different texture: smooth, rough, fuzzy, bumpy. Label each one.
Page 07
Space
Draw one strong object on a mostly empty page. Show how the empty space makes the object pop.
WHAT I LOOK FOR / 04 / 05
Expectations
Clear Element
Each sketch shows the element it is supposed to show. No guessing.
Neat and Intentional
Pages look like you cared. No scribbles. No fillers.
Short Notes
A line or two on each page explains how your drawing shows the element.
All 7 In Class
All seven pages get done in class. Color is welcome if it helps your idea.
WHAT TO SUBMIT / 05 / 05
One Clean PDF
Scan all 7 sketchbook pages into one PDF. Make sure every page is bright, in focus, and easy to read. Upload the PDF here.
Work fast. Stay focused. This is about understanding, not making a masterpiece. When you finish, click the Done button to move to Step 03.

RETO DEL CUADERNO / 01 / 05
Memoriza los Siete Elementos
Leíste la página de Elementos del Arte. Ahora tu mano tiene que aprenderlos. La forma más rápida es dibujar cada uno.
Vas a llenar siete páginas del cuaderno, un elemento por página: Línea, Forma, Volumen, Color, Valor, Textura, Espacio.
Esto no es una obra maestra. Es un estudio rápido. Enfócate en mostrar lo que cada elemento significa.

TUS SIETE BOCETOS / 03 / 05
Un Elemento por Página
Aquí hay una idea rápida de qué dibujar en cada página. Puedes escoger tu propio tema. La meta es mostrar el elemento claramente.
Página 01
Línea
Dibuja muchas líneas distintas: rectas, curvas, gruesas, delgadas, punteadas, en zigzag. Etiqueta algunas.
Página 02
Forma
Llena la página de formas planas: círculos, cuadrados, estrellas, blobs. Mantenlas planas. Sin sombra.
Página 03
Volumen
Toma una forma y hazla 3D con sombra. Un círculo plano se vuelve una esfera. Un cuadrado se vuelve un cubo.
Página 04
Color
Haz una rueda de colores o pinta áreas grandes con colores cálidos y fríos. Etiqueta cuál grupo es cuál.
Página 05
Valor
Dibuja una fila de cuadros del blanco al negro. Muestra los pasos intermedios. Esto es una escala de valor.
Página 06
Textura
Dibuja cuatro cuadros. Llena cada uno con una textura distinta: liso, áspero, peludo, con bultos. Etiqueta cada uno.
Página 07
Espacio
Dibuja un objeto fuerte en una página casi vacía. Muestra cómo el espacio vacío hace que el objeto destaque.
LO QUE YO BUSCO / 04 / 05
Expectativas
Elemento Claro
Cada boceto muestra el elemento que le toca. Nada de adivinar.
Limpio e Intencional
Las páginas se ven como que te importaron. Sin garabatos. Sin rellenos.
Notas Cortas
Una línea o dos en cada página explican cómo tu dibujo muestra el elemento.
Las 7 En Clase
Las siete páginas se terminan en clase. El color es bienvenido si ayuda a tu idea.
QUÉ DEBES ENTREGAR / 05 / 05
Un Solo PDF Limpio
Escanea las 7 páginas del cuaderno en un PDF. Asegúrate de que cada página tenga buena luz, esté enfocada y sea fácil de leer. Sube el PDF aquí.
Trabaja rápido. Mantente enfocado. Esto es para entender, no para hacer una obra maestra. Cuando termines, haz clic en el botón Done para ir al Paso 03.
